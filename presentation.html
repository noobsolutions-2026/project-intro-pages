Cybersecurity Project Management Application

Streamlining Security Assessments from Start to Finish

The Challenge

In today's complex digital landscape, managing cybersecurity projects can be a daunting task. Organizations often face:

  • Disjointed Workflows: Lack of a centralized system for project tracking.
  • Access Control Issues: Ensuring the right people have the right level of access to sensitive project data.
  • Inefficient Communication: Delays in assessment updates and project status.
  • Security Concerns: Protecting confidential project files and client information.

Our Solution

A comprehensive web application designed to bring clarity, efficiency, and robust security to your cybersecurity project management processes.

  • Centralized Platform: All projects, assessments, and files in one secure location.
  • Role-Based Access: Tailored access for Admin, Sales, and Delivery teams.
  • Automated Workflows: From assignment to project completion.
  • Enhanced Security: Built with enterprise-grade authentication and data protection.

Key User Benefits

Experience a new level of productivity and control:

  • Personalized Dashboards: Get an instant overview of your projects and pending tasks.
  • Effortless Project Management: Create, track progress, and manage project lifecycles with ease.
  • Streamlined Assessments: Assign tasks, update statuses, and monitor assessment completion.
  • Secure File Handling: Upload, store, and access all project-related documents securely.
  • Automated Updates: Projects auto-complete as assessments are finalized.

Who Uses It?

Our application caters to different roles within your organization, providing tailored functionalities:

  • Admin Users:
    • Full user and role management.
    • System configuration and oversight.
  • Sales Team:
    • Project creation and client onboarding.
    • Upload and manage sales materials.
  • Delivery Team:
    • Access and perform assigned assessments.
    • Update assessment statuses and track progress.

Security You Can Trust

Your data security is our top priority:

  • Microsoft Entra ID (Azure AD): Industry-leading authentication for secure login.
  • Role-Based Access Control (RBAC): Ensures that users only access what they need.
  • Secure Data Storage: Leverages AWS DynamoDB and S3 for highly secure and compliant storage.
  • Secrets Management: Sensitive configurations managed via AWS SSM Parameter Store.

A Glimpse Under the Hood

Built with modern, scalable, and secure technologies:

  • Backend: Node.js & Express.js for a powerful and efficient server.
  • Cloud Infrastructure: AWS (DynamoDB, S3, SSM) for reliability, scalability, and security.
  • Authentication: Passport.js with `passport-azure-ad` for seamless Entra ID integration.
  • Frontend: Vanilla HTML/CSS/JavaScript for a lean and responsive user experience.
  • Containerization: Docker for consistent development and deployment environments.
