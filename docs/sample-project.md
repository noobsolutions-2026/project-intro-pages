# Sample Project

**Intent:** Demonstrate the spec-driven generator and GitHub Pages workflow

Sample Project is a minimal demonstration of how to use the spec-driven generator to
create GitHub Pages content for small projects and tutorials.


## Key features

- Spec-driven generation
- Simple agent scripts for generation, testing, and deployment

## Usage

Run `npm run generate` to produce the `docs/sample-project.md` page from this spec.
